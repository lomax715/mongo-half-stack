## GET requests to /teams
* `/teams` 
    * returns **all dogs in db**

* `teams/id`
    * returns **team with id**

## POST request to /teams
* adds team to db

## PUT request to /teams/id
* updates **team with id**

## DELETE request to /teams/id
* deletes **team with id**
